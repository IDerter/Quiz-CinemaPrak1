# [Movie Quiz - Guess the movie] - Квиз/Викторина

Увлекательная игра-викторина, разработанная на игровом движке Unity. Приложение позволяет пользователям проверять свою эрудицию, отвечать на интересные вопросы и проходить увлекательные уровни через приятный и отзывчивый графический интерфейс.

## Описание проекта

Данный проект представляет собой кроссплатформенную квиз-игру, реализующую базовые и продвинутые механики жанра викторины. Игра предлагает два совершенно разных формата прохождения, чтобы удовлетворить как любителей долгих испытаний, так и тех, кто предпочитает достигать конкретных целей. Проект оптимизирован для мобильных устройств и веб-браузеров, демонстрируя работу с UI в Unity, управление состояниями игры и работу с базами данных вопросов.

Игра уже опубликована и доступна для пользователей на популярных площадках!

**Основные возможности:**
- **Два уникальных игровых режима:** - **Классика:** бесконечный режим на выживание и проверку общих знаний. Содержит огромную базу — **около 500 вопросов** на самые разные темы.
  - **Приключение:** сюжетно-прогрессивный режим, состоящий из **36 проработанных уровней** с возрастающей сложностью.
- Кроссплатформенная поддержка (Android и WebGL).
- Приятное визуальное и звуковое сопровождение.
- Сохранение прогресса игрока.
